# FatDown
版本v1.0.0
创建时间：2019-07-28
修改时间：2019-07-28

### 简介
葆康减脂平台

### 软件架构
Nginx 反向代理——> tomcat1,tomcat2,tomcat3
数据库云储存

### 安装教程

使用idea导入vcs -> git
自动导入依赖

### 使用说明

1. 接口使用swagger2文档[本地访问http://localhost:8088/fatdown/swagger-ui.html](http://localhost:8088/fatdown/swagger-ui.html)
备注：接口文档未在服务器，设置Nginx转发无法直接使用
